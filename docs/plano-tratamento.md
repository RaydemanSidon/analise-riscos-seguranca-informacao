# Plano de Tratamento de Riscos

| Risco | Estratégia | Ação Proposta |
|-------|------------|---------------|
| Ransomware | Mitigar | Implementar EDR e backup offline |
| Vazamento | Mitigar | Implementar DLP e criptografia |
| Falha de Backup | Mitigar | Testes trimestrais de restauração |
| Acesso Indevido | Mitigar | Implementar MFA |
